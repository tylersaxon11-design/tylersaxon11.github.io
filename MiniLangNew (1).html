Full Documentation

    

      
      
Text & Display

      

        
Title("Text")Sets the Browser Tab Name (Doesn't print to screen).

        
print("Text")Prints normal text to the screen.

        
ClearOutputRemoves all text (keeps images).

      

      
      

      
      
Variables & Math

      

        
Name = 10Sets variable 'Name' to 10.

        
Name = inputAsks the player to type something.

        
Change Name By 1Adds 1 to 'Name'. Use negative to subtract.

      


      


      
Logic & Control

      

        
Forever ... endLoops the code inside forever.

        
if Condition thenRuns code if condition is true.

        
elseif / elseAlternative branches.

        
wait(Seconds)Pauses code execution.

        
if Key Pressed thenChecks if key (Space, K) is held.

      


      


      
Images (Visuals)

      

        
Name:Draw(x, y)Draws image.

        
Name:Move(x, y)Moves image.

        
Name:FullScreenStretches to cover screen.

        
Name:Hide / :ShowToggle visibility.

        
ClearImagesRemoves all images.

      


      


      
Audio

      

        
Name:PlayPlays sound.

        
Name:StopPlayStops sound.
